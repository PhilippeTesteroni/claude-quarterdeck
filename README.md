# Quarterdeck

Command deck for your Claude Code fleet: a Windows and macOS tray app that watches
every running Claude Code session, shows live status at a glance, fires native
notifications when an agent finishes or needs you, and lets agents ask you questions
through an always-on-top popup (the answer is routed back to the session over MCP).

Everything runs locally. No telemetry, no network calls except the `127.0.0.1` MCP
endpoint.

Status: in active development. See `SPEC.md` for the locked specification and
`TASKS.md` for the build plan.
